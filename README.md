# Banking System

A console banking system I built to practice Java.

## Features

- Savings account (4% interest)
- Current account (1% interest, $1000 overdraft)
- Deposit, withdraw, view balance
- Transaction history
- Data saves to CSV files

## How to Run

```bash
git clone https://github.com/fretdev/banking-system.git

cd banking-system
mvn compile
mvn exec:java -Dexec.mainClass="com.bank.ui.Main"

Technologies
-Java 17

-Maven
